**Deterministic Password Generation** — генератор детерминированных паролей

**Используется:** Argon2id, HKDF на базе SHA-256

**Гарантируется:** Пароль содержит хотя бы один псевдослучайный символ из 
каждого набора в любой псевдослучайной позиции пароля. Т.е. по умолчанию в 
пароле всегда есть одна строчная и заглавная буква, цифра, спецсимвол из 
```specials
!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~
```

**Основная логика**: Есть мастер ключ (`key`), который генерируется из имени 
мастера (`master`) и мастер ключа (`password`) с помощью Argon2id. Из мастер
ключа генерируется 128 байт энтропии для пароля конкретного логина, которые 
рендерятся в пароль используя алфавит.

Сам Argon2id трудозатратен, поэтому можно переиспользовать ключ. 

**Концепции:** 
- `master` должен быть достаточно уникальным. Вы можете использовать любую
удобную строку: ваш email, ФИО, псевдоним и т.д. _Параметр используется, чтобы
предотвратить глобальный перебор паролей и вынудить злоумышленика подбирать 
пароли под каждого пользователя отдельно._
- `password` мастер пароль.
- Вы можете востановить любой пароль зная `master` и `password`. Это даёт 
максимальную переносимость, не возможность потерять пароль, но и означает, что 
злоумышленику достаточно знать мастер пару чтобы знать все ваши пароли

### Установка:
**Примечание:** весь CLI я навайбкодил. Но алгоритмы я писал лично, 
обсуждая с Gemini

**Без установки репозитория**
```shell
go install github.com/myau-def/detpwg/cmd/detpwg@latest

```
Должно быть доступно глобально как `detpwg`

Проверьте что `$GOPATH/bin` установлен в вашем `$PATH`

---

**С установкой репозитория**
```shell
git clone https://github.com/myau-def/detpwg.git
cd detpwg
go build -o detpwg ./cmd/detpwg
```

Доступно как `./detpwg`

### Использование:
Вы можете узнать из `detpwg --help`

Примеры использования
```shell
# Генерируем ключ
$ detpwg key master password
# 276b0552a8bd3655660c602ea9a9b470cc08f8ac9e8727ac5c94b014784e3f44
# Сгенерированный ключ можно экспортировать в переменную и использовать 
# переменную 

# Генерация пароля из ключа (супер быстро)
$ detpwg fkey \
    276b0552a8bd3655660c602ea9a9b470cc08f8ac9e8727ac5c94b014784e3f44 \
    service login -l 32 
# 4#"VzhEb{6~3~8E>i:BLnn\bl5^KqyS#

# Генерация пароля из мастер пары имени и пароля (медленне)
$ detpwg fpw master password service login -l 32 
# 4#"VzhEb{6~3~8E>i:BLnn\bl5^KqyS#
```
